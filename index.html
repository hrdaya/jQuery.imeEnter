jQuery.mcInputEvent

jQuery.mcInputEventはIME入力中にinputイベントを発火させたくない時に使用するプラグインです。

View the GitHub project

概要

jQuery.mcInputEventについて

特徴

  • 「input」の代わりに「mcinput」というイベントが発火します。
  • IME入力中にイベントは発火しません。
  • IEでは contenteditable="true" を設定したエレメントではinputイベントが発火しませんが、mcinputは発火します。
  • IE9ではバックスペースで文字を削除した時等にinputイベントが発火しないバグがありますが、mcinputでは発火します。

確認済みのブラウザ

「windows7 64bit」で下記のブラウザを使用して確認しています

  • Chrome
  • Firefox
  • Internet Explorer11
  • Internet Explorer9(Internet Explorer11でエミュレーション)
バグや改善項目がありましたら、GitHubのissueにあげてもらえると助かります。

はじめ方

プラグインの読み込み・適用

プラグインの読み込み

<script src="path/to/jquery.mcInputEvent.js"></script>

エレメントへプラグインの適用

$(".mcinput").mcInputEvent();

プラグインイベントの再適用

$(".mcinput").mcInputEvent("on");

プラグインイベントの一時不使用

$(".mcinput").mcInputEvent("off");

プラグイン適用の破棄

$(".mcinput").mcInputEvent("destroy");

プラグインイベントの強制発火

$(".mcinput").mcInputEvent("fireEvent");

イベントの捕捉

$(".mcinput").on("mcinput", function () {
    // ここに何かの処理
});

on/offするならこちら

$(document).on("mcinput", ".mcinput", function () {
    // ここに何かの処理
});

デモ

<script type="text/javascript">

  $(function () {
    "use strict";
    $(".mcinput").mcInputEvent();

    $(document).on("mcinput", ".mcinput", function () {
      var inputStr = $(this).val();
      var preStr = $("#pre").text();
      // preにinputの内容を追記していきます
      $("#pre").text(preStr !== "" ? preStr + "\n" + inputStr : inputStr);
    });
  });

</script>

<input type="text" class="mcinput">
<pre id="pre"></pre>

上記の「input」に文字を入力するとイベント発火時に下記に「input」の内容が追記されます
